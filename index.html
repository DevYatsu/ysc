ysc

A lightweight scripting language with a register-based VM, NaN-boxed values, closures, and async/await — no try/catch, no semicolons, no nonsense.

Get Started Try Online

📘Language Guide

Functions, closures, control flow, modules, error handling, and async/await.

📚Standard Library

print, str, len, time, sleep, fetch, and built-in string/list/object methods.

▶Playground

Run ysc in the browser with syntax highlighting, AST viewer, and bytecode disassembler.

⚙️GitHub

Source code, issues, and contributing — built in Rust, compiled to WASM.

Quick Example

fun fib(n) {
  if n < 2 {
    ret n
  } else {
    ret fib(n - 1) + fib(n - 2)
  }
}

print("fib(10) = " + fib(10))

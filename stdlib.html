Standard Library

Built-in Functions

FunctionDescriptionExample
print(...) Output values to stdout, separated by spaces print("hello", 42)
str(val) Convert any value to its string representation str(42) → "42"
len(val) Length of string, list, or object len("hi") → 2
time() Current Unix timestamp in seconds (float) t = time()
sleep(ms) Pause execution (returns a Promise) await sleep(1000)
fetch(url) HTTP GET request (returns a Promise, async) body = await fetch(url)
spawn(fn, ...) Spawn a function call on the event loop p = spawn("fun", 1, 2)
